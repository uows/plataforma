¿Qué estoy aprendiendo?

En este módulo estoy aprendiendo la estructura básica de una página web, el uso de etiquetas semánticas, Bootstrap para diseño responsivo y cómo organizar correctamente el contenido para una mejor experiencia de usuario.

Dato adicional: Las etiquetas semánticas ayudan a mejorar la organización y el posicionamiento en buscadores (SEO). Bootstrap nos permite crear interfaces responsivas de manera eficiente.
